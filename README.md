# TraderIQ – AI Powered Trader Intelligence Platform

## Problem Statement

Traders often make decisions based on emotions, market sentiment, or incomplete analysis. Most retail traders do not have access to tools that evaluate their trading performance and provide personalized insights. As a result, inconsistent decision-making can lead to unnecessary losses.

## Solution

TraderIQ is an AI-powered web application that analyzes trader performance using historical trading metrics and market sentiment. The system predicts potential trading outcomes and provides AI-generated trading advice to help users make more informed decisions.

The application combines machine learning predictions with a simple dashboard where users can register, log in, analyze trading performance, and review prediction history.
